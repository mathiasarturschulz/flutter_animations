# flutter_animations

Um simples projeto flutter com exemplos de estilos e animações

## Getting Started

Criar o projeto Flutter com os arquivos iniciais

`flutter create start_flutter`

Após criar os arquivos é necessário selecionar e inicializar o dispositivo que irá rodar a aplicação

Para rodar a aplicação: 

`Run > Run Without Debugging` ou `Shift+F9`
